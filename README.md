# Spring Board Project

## 💡 About Project
- Spring 프레임워크를 활용해보기 위한 목적으로 게시판 CRUD 프로젝트를 진행했습니다.

## 🎲🎯Functions
- 게시글을 등록 및 수정, 삭제할 수 있습니다.
- 게시글에 댓글을 쓰거나 삭제할 수 있습니다.
- 게시글에 파일을 첨부하거나 다운로드할 수 있습니다.
